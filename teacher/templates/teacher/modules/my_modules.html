Guide de Gestion des Modules

Tout ce que vous devez savoir sur la gestion de vos modules d'enseignement

Code d'Inscription

Les étudiants utilisent le code du module pour s'inscrire automatiquement à vos cours. Partagez ce code avec vos étudiants.

Créer des Modules

Créez vos propres modules d'enseignement. Ils nécessitent une approbation administrative avant activation.

Gestion des Étudiants

Suivez les inscriptions et gérez les projets de vos étudiants depuis chaque module. Voir les détails de chaque étudiant.

Créer des Devoirs

Créez des devoirs directement depuis vos modules. Choisissez entre assignation directe ou choix multiple pour vos étudiants.

Évaluations

Accédez rapidement aux projets nécessitant une évaluation depuis le centre d'évaluation ou la page du module.
